## Reference topics

Handbook-style templates aligned with Competitive Programmer's Handbook / CP-Algorithms staples:

- **data-structures/** тАФ Fenwick (1D/2D), segtree (sum/lazy/persistent), DSU (+rollback), sparse table, treap (+implicit), Li Chao, wavelet, binary/XOR trie, sqrt decomp
- **graphs/** тАФ BFS/DFS, shortest paths, MST, SCC, bridges/arts, block-cut, flows/matching (Dinic, MCMF, Kuhn, HopcroftтАУKarp, Hungarian), trees (LCA, HLD, centroid, Euler tour, reroot via patterns), 2-SAT, Eulerian path, difference constraints
- **math/** тАФ modint, combinatorics/Lucas, sieve/M├╢bius, CRT (+extended), BSGS, TonelliтАУShanks, XOR basis, Gaussian elim, FFT/NTT, floor sum, geometry (2D, closest pair, half-planes), factoring
- **patterns/** тАФ binary/ternary search, two pointers, sliding window, LIS, MoтАЩs, CHT, DP family (digit/bitmask/SOS/rerooting), coordinate compression, contest I/O
- **strings/** тАФ KMP, Z, Manacher, hashing, trie, AhoтАУCorasick, SA/SAM/suffix tree, Lyndon

## Compile

Every `.cpp` under `coding/` (flags from this README):

```bash
# Local / CI (Docker)
docker build --target coding -t interview-coding -f .github/docker/Dockerfile .
docker run interview-coding

# Or directly
./.github/scripts/compile-coding.sh
```

Single file:

```bash
g++ -std=c++17 -O2 -Wall reference/graphs/shortest-path.cpp
g++ -std=c++17 -O2 solved/hacker-cup/2021-qualification-consistency-a1.cpp
```

Format: `clang-format -i **/*.cpp`
